## 📚 Sobre

O projeto em questão é um tocador de som que foi desenvolvido com javascript.


O objetivo foi para praticar o uso de condições e lógicas de programação.


## 🚀 Tecnologias utilizadas

- ReactJS
- TypeScript
- Sass
- Css Modules
